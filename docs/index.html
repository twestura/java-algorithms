Java Algorithms, Data Structures & Other Notes

Notes on algorithms, with supporting code and examples written in Java. Their level ranges from a second-semester programming course to an introductory or more advanced algorithms course. They are intended more as a reference than as a tutorial, and should be useful for anyone wishing to review for programming interviews or improve at competitive programming problems.
